# 📌 함수형 프로그래밍

- 프로그램을 계산으로 보는 것.
- 무엇(What)을 하는가에 초점.
- 객체지향적인 사고가 함수형에 대입이 가능.

### ✅ 함수도 1급 객체

- 변수를 저장 할 수 있음.
- 매개변수로 전달을 할 수 있음.
- 반환 값이 될 수 있음.
- 고유하게 구별이 가능.
- 동적으로 할당이 가능.

```js
const adder = (base) => (v) => base + v;
const add3 = adder(3);

console.log(add3(7)); //10
```

### ✅ Arrow Function

- new 연산자를 사용할 수 없음.
- 메모리가 해제되지 않음.
